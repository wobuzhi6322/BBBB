How it works

후원은 계좌로 받고, 화면 반응은 계이득이 처리합니다

은행 알림을 직접 결제처럼 꾸미지 않습니다. 방송 PC가 받은 입금 알림을 읽고, 금액과 이름을 정리한 뒤 미리 저장한 OBS 리액션으로 연결합니다.

  1. 1입금 알림 수신

    은행 앱 알림이 Windows Phone Link를 통해 PC에 도착합니다.

  2. 2이름과 금액 정리

    알림 문장에서 입금자와 금액을 추출하고 중복 알림을 걸러냅니다.

  3. 3시그니처 선택

    금액, 후원자, 랭킹, 누적 조건에 맞는 규칙을 선택합니다.

  4. 4OBS 출력

    브라우저 소스로 시그니처, 영상, 음악, 벽지를 방송 화면에 띄웁니다.
